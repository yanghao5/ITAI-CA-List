# ITAI-CA-List
Information Technology Application Innovation CA Supplier List

# DV OV 和 EV 证书

| 项目 | DV 证书 | OV 证书 | EV 证书 |
| --- | --- | --- | --- |
| 验证内容 | 域名控制 | 域名 + 公司 | 域名 + 公司 + 法律和运营状态 |
| 安全性 | 基础 | 中等 | 最高 |
| 浏览器显示 | 小锁图标 | 小锁图标 + 可查看组织名 | 旧浏览器绿色地址栏（现代已取消） |
| 签发速度 | 快速 | 1\~3 天 | 2\~7 天 |
| 适合人群 | 个人 | 企业 | 金融、电商等对信任要求高的机构 |

# 国密 SSL/TLS Cert CA List

国密 SSL 证书，并不被主流操作系统（包括firefox）使用.

它们被信创操作系统（银河麒麟、统信 UOS、OpenEuler）或国密浏览器（奇安信浏览器、360安全浏览器）采用.
